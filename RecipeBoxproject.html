RecipeBox - an Android app

Tasked with creating an Android app, we went beyond the standard weather apps and calculators to make a unique tool that we would actually use in our lives. RecipeBox uses computer vision to detect different foods in the user's picture and returns a list of recipes that can be made using the foods detected. Find it on GitHub.

Aaron Alberg, Davis Keene

My Role: frontend, UI/UX design, logo design

Skills/Tools: Android Studio (Java), Heroku, PowerPoint

Development

The photo taken or uploaded by the user is processed using the clarifai API, which returns foods it indentifies. This list is then passed to the Recipe Puppy API to return recipes for the user.

The frontend was developed in Java and uses Gson to accept data from the backend written in Python. I went with a minimalist logo to maintain consistency with the clean layout of the app.

Moving Forward

Future plans for Recipe Box currently include personalization options for users, allowing them to create accounts and save favorite recipes. A port to iOS is also a possibility once feature set has been expanded and the app is more stable.

Recipe Box was chosen by peers as one of the the most impressive projects in a class of over 800 students.
